Версия v13: добавлены недостающие детальные ЗИЛы в Case Library / Vehicle Art. Картинки оптимизированы в WebP.

КАК ЗАЛИВАТЬ НА GITHUB

1. Распакуй ZIP.
2. Не загружай сам ZIP-файл.
3. В репозиторий deniss-portfolio загружай содержимое папки: index.html, en.html, ru.html, style.css, script.js, data, assets.
4. Если GitHub через браузер ругается, сначала загрузи файлы HTML/CSS/JS/data, потом assets/images частями.

Эта версия оптимизирована: картинки переведены в .webp, чтобы GitHub легче принимал загрузку и сайт быстрее работал.
